This project presents a Risk-Aware Adaptive Hybrid Deep Learning Framework for financial time-series prediction, aiming to improve both predictive accuracy and financial risk management. The study integrates Convolutional Neural Networks (CNN), Bidirectional Long Short-Term Memory (BiLSTM), and attention mechanisms to capture complex temporal patterns in financial data. A novel RiskAwareLoss function is introduced to incorporate downside risk into the learning process, addressing the limitations of traditional loss functions that focus only on prediction accuracy. The model is evaluated using S&P 500 and VIX datasets with comprehensive feature engineering and a two-stage training approach. Experimental results demonstrate that the proposed hybrid framework enhances directional accuracy and risk-adjusted performance, while effectively reducing drawdowns in volatile market conditions. Overall, the research contributes to bridging the gap between deep learning-based forecasting and practical financial risk management.
